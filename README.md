# GitHub Pages 臨時頁面

這個資料夾已經整理成可直接放到 GitHub Pages 的最小結構：

- `index.html`：網站首頁
- `.nojekyll`：避免 GitHub Pages 套用 Jekyll 處理

## 發布方式

1. 建立一個新的 GitHub Repository。
2. 把這個資料夾內容上傳到 repository 根目錄。
3. 如果你要用個人首頁，repository 名稱請設成 `<你的 GitHub 帳號>.github.io`。
4. 如果只是臨時專案頁，repository 可自訂名稱，再到 GitHub 的 `Settings > Pages` 開啟部署。

## 網址規則

- 個人首頁：`https://<你的 GitHub 帳號>.github.io/`
- 專案頁：`https://<你的 GitHub 帳號>.github.io/<repository 名稱>/`
